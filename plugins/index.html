PotaTOECraft Plugins

These are the amazing plugins we use to make the server awesome!

1. ChestShop – Create and manage player shops to buy and sell items.
2. CoreProtect – Protect the world from griefing with rollback features and logging.
3. Essentials – Core plugin for basic server commands like teleportation, homes, and more.
4. EssentialsDiscord – Connects the Minecraft server with Discord for easy communication.
5. EssentialsXChat – Customizes chat formatting to make it cleaner and more organized.
6. EssentialsXSpawn – Manages spawn locations, teleportation, and new player setups.
7. LuckPerms – A powerful permissions plugin to manage user ranks and permissions.
8. MyCommand – Customizes commands and creates unique, personalized commands for the server.
9. TAB – Customizes the player list and display features to enhance visibility and user interaction.
10. TogglePvp – Gives players the option to enable or disable PvP in specific areas.
11. Vault – Essential plugin for economy, permissions, and other related management.
12. ViaVersion – Allows players from different Minecraft versions to connect to the server.
13. WorldEdit – Powerful in-game editing tool for building large structures and world modifications.
